# 项目商业与运营概览（v1.0 MVP）

面向门店/客服的自动化运营中台：飞书表格做数据库，Python + PC 微信 RPA 做执行，解决“加好友累、备注乱、忘回访、撞单、数据易丢”等问题。深圳已有 33 家门店场景，适合连锁/单店快速落地。

## 业务痛点与价值
- 资产流失风险：客户信息散落在个人微信/脑子里，员工离职带走客源。
- 虚拟号壁垒：美团/抖音虚拟号无法沉淀私域。
- 被动服务：未成交/流失客没人跟，靠记忆容易漏。
- 昵称割裂：微信昵称频繁变，无法对齐真实客户。

**价值点：**
- 备注统一 = 可追溯：自动改备注为“手机号-姓名”，昵称再变也能精确定位。
- 自动化补课：加友、欢迎语、预约冲突检查、到店提醒、回访提醒全流程机器人代劳。
- 数据安全：客户档案/预约流水/排班都在飞书，多人协同且可交接。

## 核心功能清单
1) 微信加好友与备注  
   - 自动加友：前台录入手机号，工作机自动搜索并发送申请。  
   - 自动改备注：加友后改为“手机号-姓名”，永远可索引。  
   - 被动建档：扫码加店员时，自动问手机号并存入表格。

2) 自动接待与提醒  
   - 自动发欢迎语：通过验证即发价目表+定位。  
   - 防撞单预约：写预约时自动检查技师该时段是否空闲。  
   - 扫码预约/排班查看：客户可扫码预约，系统核验排班。  
   - 到店提醒：预约前 1 小时提醒前台/技师准备。

3) 自动回访与流失预警  
   - 按项目计算回访日：如做脸后第 3 天询问过敏、洗发后第 7 天询问养护。  
   - 流失预警：超过 45 天未到店，推送急办任务给店长跟进。

4) 数据统计与看板  
   - 转化率：电话 -> 加微 -> 到店。  
   - 员工业绩：技师接单/收入/饱和度。  
   - 渠道效果：美团/抖音/点评的客单与贡献。  
   - 客群分析：低价/办卡分布。

5) 资料安全与交接  
   - 数据归属老板：全部在飞书表格。  
   - 一键交接：员工离职可迁移其负责客户给新员工。

## 现状与核心痛点（门店视角）
1. 资产流失风险：客户信息散落在个人微信、通话记录和大脑里，一旦微信记录丢失，客户资源随之流失（缺乏核心数据库）。  
2. 虚拟号壁垒：美团/抖音带来的客流是加密虚拟号，无法直接转化为私域流量，导致一次性生意占比过高。  
3. 被动等待：未成交/流失客户缺少自动唤醒机制。  
4. 身份割裂：微信昵称与真实客户/预约记录无法对应，服务错位。

## 解决方案：Feishu CRM + 微信自动化中台
不采购昂贵 SaaS，基于飞书多维表格（后端数据库）与自研 Python 自动化脚本，打造轻量定制化运营系统。

### 核心模块
1. 中央数据库（飞书多维表格）
   - 客户档案表：手机号唯一身份证，记录消费习惯、会员等级、微信绑定状态。
   - 预约流水表：防撞单，记录每笔服务状态。
   - 人员排班表：管理技师工时，自动关联预约。
   - 可视化看板：今日客流、转化率、员工业绩排行。

2. 身份统一协议（备注强制）
   - 逻辑：所有加到微信的客户，自动改备注为 `[手机号]-[姓名]`。
   - 效果：无论客户怎么改名，系统都能通过手机号精准识别。

3. 自动化工作流（RPA）
   - 自动加友：前台录入手机号，电脑自动唤醒微信添加。  
   - 被动建档：顾客扫码加微信，机器人自动询问手机号并录入飞书。  
   - 欢迎语/门店指引自动发送（支持多图+文本）；预约前 1 小时提醒；回访/流失预警。

## 实施路径（MVP，两阶段）
**阶段 1：数据奠基期（第 1 周）**  
- 目标：跑通流程，养成习惯（可人工+飞书表，不启脚本）。  
- 场景 A：接听虚拟号 -> 话术问手机号 -> 前台填飞书表单 -> 生成待添加任务。  
- 场景 B：到店/咨询 -> 填预约卡（飞书链接） -> 自动建档。

**阶段 2：自动化提效期（第 2 周起）**  
- 目标：解放人力，引入脚本。  
- 上线：自动添加、自动备注、预约前提醒、回访/流失预警。

## 预期收益
1. 资产安全：客户数据归店铺，员工离职不丢客。  
2. 效率提升：每天节省 1-2 小时重复操作。  
3. 数据决策：随时看报表，识别渠道/业绩/客群。  
4. 流量召回：自动关怀流失客，预期提升 10%-20% 回头率。

## 风险与配合要求
- 硬件：前台需 Windows 电脑，工作时段保持开机联网。  
- 账号：使用店铺专用工作微信（避免私人号）。  
- 执行：前台必须严格执行“来电即录入”SOP，数据准确性决定智能程度。

## 提问：能赚钱吗？
对于 33 家门店规模，若客单和复购符合常规美业/到店业务场景：
- 盈利杠杆：减少流失 + 提升复购 + 提升人效。  
- 典型回收：节省人力与漏单损失（加友/回访自动化）、提升到店率（预约提醒）、提升复购（回访/流失关怀），可在数周内覆盖部署与维护成本。
